This is RP2040 Zero 8x11 matrix\
The matrix is controlled using Window Dynamic Lighting Protocol\
When lighting is not in use, Conway's game of life is played as the animation\
Currently only tested on Win11 Dynamic Lighting Control and SignalRGB\
LEDS used is WS2812B wired in serpentine\
The brightness is low because its fully usb powered\
Higher brightness need external power for the leds
